Class macgyer\yii2materializecss\widgets\NavBar

All Classes | Properties | Methods
Inheritancemacgyer\yii2materializecss\widgets\NavBar » macgyer\yii2materializecss\lib\BaseWidget » yii\base\Widget
Uses Traitsmacgyer\yii2materializecss\lib\MaterializeWidgetTrait

NavBar renders a navbar HTML component.

Any content enclosed between the \yii\base\Widget::begin() and \yii\base\Widget::end() calls of NavBar is treated as the content of the navbar. You may use widgets such as Nav or \yii\widgets\Menu to build up such content. For example,

use macgyer\yii2materializecss\widgets\NavBar;
use macgyer\yii2materializecss\widgets\Nav;

NavBar::begin(['brandLabel' => 'NavBar Test']);
echo Nav::widget([
    'items' => [
        ['label' => 'Home', 'url' => ['/site/index']],
        ['label' => 'About', 'url' => ['/site/about']],
    ],
]);
NavBar::end();

See also http://materializecss.com/navbar.html.
