Class macgyer\yii2materializecss\widgets\grid\GridView

All Classes | Properties
Inheritancemacgyer\yii2materializecss\widgets\grid\GridView » yii\grid\GridView

The GridView widget is used to display data in a grid.

It provides features like sorting, paging and also filtering the data.

A basic usage looks like the following:

<?= GridView::widget([
    'dataProvider' => $dataProvider,
    'columns' => [
        'id',
        'name',
        'created_at:datetime',
        // ...
    ],
]) ?>

The columns of the grid table are configured in terms of yii\grid\Column classes, which are configured via yii\grid\GridView::$columns.

The look and feel of a grid view can be customized using the large amount of properties.
